# Фінальний проєкт: Алгоритми та структури даних

Цей проєкт є комплексною роботою, що об'єднує сім різнопланових завдань: від маніпуляцій з лінійними списками до статистичного моделювання методом Монте-Карло. Для зручності користувача всі завдання інтегровані в єдину систему керування.

---

## Початок роботи та встановлення

Перш ніж запускати проєкт, необхідно переконатися, що у вашому середовищі встановлені всі необхідні бібліотеки. Проєкт використовує:

* **colorama** — для стилізації термінала.
* **networkx** та **matplotlib** — для побудови графіків та візуалізації структур.
* **pulp** — для розв'язання задач оптимізації.

---

## Встановлення залежностей

Відкрийте термінал у кореневій папці проєкту та виконайте команду:

```bash
pip install -r requirements.txt

---

## Використання головного файлу (`main.py`)

Ми розробили файл `main.py` як центральний вузол керування проєктом. Вам не потрібно шукати та запускати кожен скрипт окремо — все доступно через інтерактивне меню.

### Як запустити проєкт:

1.  **Запуск меню:** У терміналі введіть команду:
    ```bash
    python main.py
    ```
2.  **Навігація:** Після запуску ви побачите список усіх доступних завдань. Введіть номер завдання (наприклад, `5`) і натисніть **Enter**.
3.  **Взаємодія:**
    * Для графічних завдань (Дерево Піфагора, візуалізація купи тощо) відкриється окреме вікно.
    * **Важливо:** Щоб повернутися до головного меню після перегляду графіка, просто **закрийте вікно** візуалізації.
4.  **Вихід:** Щоб завершити роботу з проєктом, оберіть пункт `0`.

## гляд завдань у меню

Програма `main.py` автоматично керує шляхами до файлів, надаючи доступ до наступних модулів:

| № | Назва завдання | Опис функціоналу |
| :--- | :--- | :--- |
| **1** | **Однозв'язний список** | Демонстрація реверсу та сортування вставками. |
| **2** | **Дерево Піфагора** | Побудова фрактала (користувач вказує рівень рекурсії). |
| **3** | **Алгоритм Дейкстри** | Пошук найкоротшого шляху в міській мережі. |
| **4** | **Візуалізація купи** | Перетворення лінійного масиву на графічну бінарну купу. |
| **5** | **Обходи дерева** | Візуалізація DFS та BFS з динамічною зміною кольорів. |
| **6** | **Оптимізація виробництва** | Розв'язання задачі лінійного програмування. |
| **7** | **Інтегрування Монте-Карло** | Порівняння статистичного обчислення з аналітичним. |

## Усунення несправностей

* **Помилка імпорту:** Якщо ви бачите `ModuleNotFoundError`, переконайтеся, що ви виконали крок зі встановлення через `requirements.txt`.
* **Графічне вікно не з'являється:** Переконайтеся, що ваш Python має доступ до графічного інтерфейсу (наприклад, на серверах без монітора бібліотека `turtle` може не працювати).
* **Шляхи:** Файл `main.py` розрахований на запуск саме з кореневої папки проєкту для коректного імпорту підмодулів.

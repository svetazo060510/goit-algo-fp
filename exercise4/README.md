# Завдання 4: Візуалізація бінарної купи

Це завдання фокусується на перетворенні лінійного представлення структури даних (масиву) у візуальну ієрархічну структуру (бінарне дерево).

---

## Логіка побудови дерева з купи

Бінарна купа зазвичай зберігається у вигляді масиву для економії пам'яті. Щоб візуалізувати її як дерево, ми використовуємо математичні властивості індексів.

Для вузла з індексом $i$:

* **Лівий дочірній вузол** знаходиться за індексом $2i + 1$.
* **Правий дочірній вузол** знаходиться за індексом $2i + 2$.

Функція `build_tree_from_heap` рекурсивно проходить по масиву, створюючи об'єкти `Node` та зв'язуючи їх відповідно до цих правил.

---

## Технічні деталі реалізації

* **NetworkX & Matplotlib:** Використовуються для генерації графа та його малювання. Координати вузлів обчислюються динамічно, щоб дерево виглядало симетричним.
* **UUID:** Кожен вузол отримує унікальний ідентифікатор. Це критично важливо, оскільки в купі можуть бути однакові значення ключа, і без унікальних ID **NetworkX** міг би об'єднати різні вузли в один.
* **Модульність:** Всі налаштування (кольори, розміри, дані) винесені в `config.py`, що дозволяє легко змінювати вигляд дерева без втручання в алгоритм.

## Використання

1. **Запустіть візуалізацію:**

    ```bash
    python heap_visualization.py
    ```

2. **Перегляд:** Відкриється вікно бібліотеки `matplotlib` із зображенням купи.
3. **Завершення:** Закрийте вікно `matplotlib`, щоб завершити роботу алгоритму та повернутися до головного меню проєкту.

## Висновки

Проєкт наочно демонструє безпосередній зв'язок між компактним **лінійним представленням** купи в пам'яті комп'ютера (масив) та її **логічною деревоподібною структурою**. Така візуалізація спрощує розуміння алгоритмів сортування купою (Heapsort) та роботу пріоритетних черг.
